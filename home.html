Informações básicas

Visando facilitar o acesso à dados abertos do governo para a comunidade de desenvolvedores o Tribunal de Contas da União criou a plataforma AppCívico, que fornece através de uma API de serviços REST informações abertas

Funcionalidades disponíveis

  • Listar escolas
  • Buscar escola pelo identificador
  • Buscar escola pelo nome
  • Buscar avaliações de uma escola selecionada
  • Buscar avaliações anuais de uma escola selecionada
  • Buscar avaliações anuais pelo tipo de uma escola selecionada
  • Buscar a media total das avaliações de uma escola selecionada
  • Buscar a media das avaliações anuais de uma escola selecionada
